# Information Security Lab

This repository contains Java implementations of practical experiments performed as part of the **Information Security Laboratory** course.

The experiments cover classical cryptography, modern encryption algorithms, authentication techniques, hashing, digital signatures, network security, and secure communication protocols.

Each practical includes:
- Java source code
- Theory and concept explanation
- Algorithm
- Compile and run instructions
- Sample output
