# 🔐 Information Security Laboratory

This repository contains Java implementations of practical experiments performed as part of the **Information Security Laboratory** course.

The experiments cover both classical and modern cryptographic techniques, including encryption algorithms, hashing, authentication mechanisms, digital signatures, key exchange protocols, and secure communication.

Each practical is organized in a separate folder and includes:

- ☕ Java source code
- 📖 Theory and concept explanation
- 📝 Algorithm
- ▶️ Compile and run instructions
- 💻 Sample output
